Here’s what makes it better than typical hackathon entries:

---

# 1. Dual‑Mode Intelligence (Online + Offline)
Most entries rely **only on cloud AI**.
If the evaluator:
- has no API key  
- has no internet  
- or the API rate‑limits them  

-> Their project breaks.

TaurgoVision never breaks:

- If Gemini is available - uses Gemini Vision  
- Else if OpenAI is available - uses GPT‑4 Vision  
- Else - uses offline CV with overlays  

This flexible architecture is extremely robust and works in all environments.

- iPhone photos

This is a direct usability advantage.
